Данный сайт для интернет магазина "New-Polynom" является семестровой работой по предмету "Фронтенд и бэкенд разработка". В разработке принимали участие студенты 2-го курса РТУ МИРЭА по направлению "Информационные системы и технологии (Фуллстек)" Вишняков Матвей, Калинин Данила и Щелкунов Алексей.

Чем занимались каждый каждый из разработчиков: Щелкунов Алексей - разработал базу данных для данных учетных записей и маршрутизацию Калинин Данила - Принимал участие в разработке дизайна сайта, написании основного кода на HTML и свойств для классов на CSS. Вишняков Матвей - Принимал участие в разработке дизайна сайта, написании основного кода на HTML и свойств для классов на CSS, а так же в написании BackEnd'a для некоторых функций.
